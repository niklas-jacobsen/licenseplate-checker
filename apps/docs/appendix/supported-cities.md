# Supported Cities

The following cities have confirmed working workflows. Each city's reservation website has been tested and a workflow can be created using the [workflow builder](/frontend/builder).

| Code | City | Website |
|------|------|---------|
| COE | Coesfeld | [kreis-coesfeld.de](https://app.kreis-coesfeld.de/wkz/?LICENSEIDENTIFIER=coe) |
| D | Düsseldorf | [service.duesseldorf.de](https://service.duesseldorf.de/wunschkennzeichen) |
| DD | Dresden | [dresden.de](https://wunschkennzeichen.dresden.de/wkz/?LICENSEIDENTIFIER=dresden) |
| K | Köln | [i-kfz.regioit.de](https://i-kfz.regioit.de/i4/ikfz4/?LICENSEIDENTIFIER=koeln_stadt) |
| L | Leipzig | [leipzig.de](https://kennzeichen.leipzig.de/wkz/?LICENSEIDENTIFIER=leipzig_stadt) |
| MS | Münster | [stadt-muenster.de](https://www.stadt-muenster.de/wkz/?LICENSEIDENTIFIER=ms) |
| N | Nürnberg | [nuernberg.de](https://online-service2.nuernberg.de/wkz) |
| P | Potsdam | [egov.potsdam.de](https://egov.potsdam.de/wkz) |
| PM | Potsdam-Mittelmark | [potsdam-mittelmark.de](https://ekol.potsdam-mittelmark.de/wkz) |
| W | Wuppertal | [wkz.wuppertal.de](https://wkz.wuppertal.de/wkz) |
| WB | Wittenberg | [wittenberg.de](https://onlinezulassung.landkreis-wittenberg.de/verkehr-igv/servlet/Internetgeschaeftsvorfaelle?AUFRUF=WKZ) |

::: info
Cities like Berlin and Hamburg that use the recently introduced `behoerden-serviceportal.de` portal are not yet supported due to the complexity of their multi-step reservation process and the requirement to authenticate.
:::
